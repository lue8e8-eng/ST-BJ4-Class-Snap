# 📅 伸動保健室｜互動排課月曆 (Course Calendar)

這是一個專為 **伸動保健室** 團隊（Charles & Ollie）量身打造的互動式排課工具。旨在簡化每日課程與休息時間的管理，並提供一鍵匯出美觀行事曆的功能。

## ✨ 核心功能
* **動態月曆視圖**：自由切換年份與月份，自動計算每月天數與星期排列。
* **直覺編輯介面**：點擊任意日期即可快速設定行程，操作簡單直覺。
* **狀態標誌識別**：
    * **上課 (藍色)**：顯示「課」字印章，並標註上課老師（Charles / Ollie）。
    * **休息 (紅色)**：顯示「休」字印章，畫面簡潔明瞭。
* **人員管理系統**：支援複選人員，並具備專屬顏色標籤（Charles: 藍色 / Ollie: 綠色）。
* **一鍵圖片匯出**：內建高解析度截圖技術，自動優化介面（隱藏按鈕），產出乾淨的 JPG 圖片方便在群組分享。
* **自訂月曆標題**：標題可自由編輯，預設為「伸動保健室預約月曆」。

## 🛠️ 技術棧
* **React**：前端核心框架。
* **Tailwind CSS**：現代化快速介面排版。
* **Lucide React**：精美且一致的 UI 圖示。
* **html2canvas**：將網頁元素轉換為高畫質圖片。

## 🚀 如何使用
1. **線上使用**：直接打開部署於 Vercel 的網址即可操作。
2. **排課**：點擊日期 -> 選擇類型與人員 -> 儲存。
3. **分享**：點擊左上角「下載圖示」，即可取得當月課表 JPG。
